The RMA Kinetics library provides a set of ordinary differential equation (ODE) models for simulating the production, transport, and degradation of Released Markers of Activity (RMAs) — synthetic serum reporters expressed in the brain. Each model exposes a `Model` struct/class that holds kinetic parameters and a `State` struct/class that holds the current concentrations of each tracked species. Solving a model returns a `Solution` object you can query by species name.

## Model and State pattern

Every model follows the same two-type pattern:

- **`Model`** — holds kinetic rate parameters. Constructed with sensible defaults derived from published experimental data. Call `.solve(t0, tf, dt, init_state, solver)` to integrate the ODE system over a time interval.
- **`State`** — holds the initial concentrations of each species. All fields default to `0.0`.

```python
from rma_kinetics.models.constitutive import Model, State
from rma_kinetics.solvers import Dopri5

model = Model()                  # default kinetic parameters
state = State()                  # all species start at 0 nM
solver = Dopri5()
solution = model.solve(0, 504, 1, state, solver)

plasma = solution.plasma_rma     # NDArray[float64], one value per time point
```

In Rust the pattern is identical — use `Model::default()`, `State::zeros()`, and call `.solve()` via the `Solve` trait.
