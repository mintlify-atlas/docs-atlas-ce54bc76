## What are RMAs?

Released Markers of Activity (RMAs) are synthetic serum reporters expressed in specific cell populations. When those cells are active, they secrete a detectable protein into the bloodstream. By measuring RMA concentration in a blood sample, researchers can infer underlying neural or cellular activity — without invasive procedures.

RMA Kinetics models the full pharmacokinetic trajectory of these reporters: from gene expression in the brain, through the blood-brain barrier, into the plasma. Models include effects of doxycycline-controlled gene silencing and chemogenetic (DREADD) induction.

## Installation

<Steps>
  <Step title="Install the Python package">
    Build from source using maturin:
    ```bash
    git clone https://github.com/nsbuitrago/rma-kinetics-rs
    cd rma-kinetics-rs
    uv venv
    uvx maturin develop --uv --features py -m crates/rma-kinetics/Cargo.toml
    ```
  </Step>
  <Step title="Or add the Rust crate">
    ```bash
    cargo add rma_kinetics
    ```
  </Step>
  <Step title="Run your first simulation">
    ```python
    from rma_kinetics.models.constitutive import Model, State
    from rma_kinetics.solvers import Dopri5

    model = Model()
    solution = model.solve(0, 100, 1, State(), Dopri5())
    print(solution.plasma_rma[-1])
    ```
  </Step>
</Steps>

<Note>
  The Python package on PyPI uses an older pure-Python implementation. For the latest Rust-backed bindings, build from source as shown above.
</Note>
